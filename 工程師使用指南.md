# AI 助手使用指南 — 工廠工程師版

**伺服器位置：** 高雄封測廠 — 本地 AI 推理伺服器  
**更新：** 2026-04-23

---

## 快速開始

### 1. 打開瀏覽器，輸入以下網址

```
http://192.168.1.100:8080
```

> 請向 IT 管理員確認實際的伺服器 IP 位址。所有功能都在瀏覽器內操作，**不需要安裝任何軟體**。

### 2. 登入

使用 IT 管理員提供的帳號密碼登入。第一次登入建議立即修改密碼（右上角頭像 → Profile）。

---

## 主要介面說明

登入後，你會看到 Open WebUI 的聊天介面，操作方式和 ChatGPT 類似。

### 選擇 AI 助手（模型）

左側欄位上方有模型選單，預設為 **qwen3.6-35b**（Qwen3.6-35B-A3B），這是主要使用的模型。除非管理員另有說明，維持此選擇即可。

### 開始對話

- 在下方輸入框輸入問題，按 Enter 或點選送出
- AI 回答速度約 **1-3 秒**出現第一個字，之後快速生成
- 如果 AI 的回答過長可以點選 **Stop** 停止

---

## 各 Agent 使用說明

系統內建以下 6 個專用 Agent（需透過 Dify 介面使用，管理員設定後工程師可直接點選）：

### Agent 1：製程工程師助手

**適用場景：** MW30 / MCX A10 製程相關問題

使用範例：
```
Q：MCX A10 bonding force 的標準值範圍是多少？
Q：MW30 製程中，若溫度偏高 5°C 會對良率有什麼影響？
Q：probe card PC-MCX-007 需要更換的條件是什麼？
```

### Agent 2：RCA 分析助手

**適用場景：** 良率異常，需要啟動根因分析

使用範例：
```
Q：Lot ID LOT-20260423-001 的 RCA 分析，異常現象：良率從 95% 掉到 78%
Q：請分析昨天 B 班 MW30 的異常，WAT 數據已附上
```

> 此 Agent 會呼叫後台 9-agent pipeline，分析時間約 30-45 分鐘，分析完成後自動通知。

### Agent 3：良率報告助手

**適用場景：** 生成月份趨勢報告、MRAT 報告摘要

使用範例：
```
Q：請幫我整理 2026 年 3 月的 MCX A10 良率趨勢，重點是 WAT fail 的分布
Q：生成本週良率日報格式
```

### Agent 4：BlueVelo 顧問助手

**適用場景：** 精密螺絲廠製造改善建議

### Agent 5：技術文獻搜尋

**適用場景：** 查詢論文、技術規格、設備手冊

使用範例：
```
Q：幫我找關於 copper wire bonding 在高溫環境劣化的最新研究
Q：JEDEC JEP122 的 electromigration 測試條件是什麼？
```

### Agent 6：程式碼助手

**適用場景：** Python 數據分析、自動化腳本、SQL 查詢

使用範例：
```
Q：幫我寫一個 Python 程式，讀取 CSV 格式的 WAT 數據，畫出各參數的 box plot
Q：幫我把這段 SQL 查詢最佳化：[貼上 SQL]
Q：這段 Python 程式報錯了，幫我找問題：[貼上程式碼]
```

## 實用技巧

### 附加檔案

可以直接拖曳或點選附件圖示，上傳以下格式的檔案讓 AI 分析：
- CSV、Excel（.xlsx）：數據分析
- PDF：文件閱讀與摘要
- 圖片（PNG、JPG）：圖表解讀

範例：
```
[上傳 WAT_data_April.csv]
請分析這份 WAT 數據，找出哪些參數有異常偏移，並指出可能的製程原因
```

### 多輪對話

AI 會記得同一個對話視窗內的上下文，可以持續追問：
```
Q：請說明 probe mark 的成因
A：[AI 回答]
Q：那如果是 wire bonding 的問題呢？  ← 不需要重新解釋背景
```

### 儲存對話

重要的對話會自動儲存在左側欄位，可以隨時點選回顧。對話標題可以點選重新命名，方便之後找到。

### 切換語言

AI 預設以繁體中文回答。如果需要英文回答，在問題開頭加上 "Please answer in English"。

## 效能與使用限制

| 情境 | 預期速度 |
|------|---------|
| 一般問答（< 1000 字回答）| 10-30 秒 |
| 長文件分析（PDF 報告）| 30-90 秒 |
| 程式碼生成（100 行以內）| 15-45 秒 |
| RCA pipeline（9 agents）| 30-45 分鐘 |

**同時使用者：** 系統支援 30 人以上同時使用（32K context 模式），請放心使用，不需要排隊。

---

## 注意事項

**資料安全：**
- 所有對話都在工廠內網處理，**不會傳送到外部網路**
- 不要輸入客戶機密、IP 資料、個人身分資料
- 帳號密碼請勿與他人共用

**AI 的限制：**
- AI 有時會給出不正確的答案，**重要決策請自行驗證**
- 最新的製程 SOP 以官方文件為準，AI 的答案僅供參考
- 如果 AI 的回答和你的經驗不符，請進一步追問或向資深工程師確認

## 常見問題

**Q：網頁打不開？**  
A：確認電腦在工廠內網（Wi-Fi 或有線），網址輸入 `http://192.168.1.100:8080`（注意是 http 不是 https）。若仍無法連線，請聯繫 IT。

**Q：AI 回答很慢或停住了？**  
A：可能是多人同時使用較忙碌的情況，稍等片刻或點選 Regenerate 重新生成。如果超過 5 分鐘無回應，請重新整理頁面。

**Q：忘記密碼？**  
A：聯繫 IT 管理員重設。

**Q：可以在手機上使用嗎？**  
A：可以，用手機瀏覽器輸入同樣的網址即可，介面會自動適應螢幕大小。

**Q：AI 的回答可以信任嗎？**  
A：AI 在一般問答和計算方面通常很準確，但製程關鍵決策（如調整製程參數、換料）請務必自行驗證，或由資深工程師確認。
